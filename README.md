# Cpp-Grade-Master

A fundamental C++ console application that demonstrates core programming concepts: Variables, Functions, Logic Branches, and Input/Output stream manipulation.



## 🌟 Highlights
- **Zero Dependencies:** Uses only standard C++ libraries (`iostream`, `string`, `iomanip`).
- **Parameterized Logic:** Optimized grading through reusable functions to maintain clean code.
- **Accurate Arithmetic:** Implements floating-point division for precise percentage calculations.
- **Clean Interface:** Formatted tabular output for professional-looking reports.

## 📊 Grading Logic
The system evaluates performance based on the following scale:
- **A+** : 80% and above
- **B+** : 70% - 74%
- **F** : Below 50% (Fail)

## 🛠 How to Use
1. Clone the repository.
2. Compile using any C++ compiler (e.g., `g++ main.cpp -o grading`).
3. Run the executable and follow the on-screen prompts.
